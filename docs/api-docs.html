📖 API Documentation

REST endpoints for market data, engine metrics, and predictions

Authentication

All API requests require a valid API key passed as the X-API-Key header.

GET /api/v1/market-data
# Get latest market data for all tracked assets
X-API-Key: your-key-here

Endpoints

GET /api/v1/engine/status
Current engine cycle, agent count, capital, win rate

GET /api/v1/engine/agents
Top agents with PnL, win rate, genome weights

GET /api/v1/market/prices
Latest prices for BTC, ETH, SPY, VIX, gold, forex

GET /api/v1/paper/stats
Paper trading statistics per market

API is in early access. Rate limits apply. Get a key from the registration page.
